Portfolio

Gift smart

GiftSmart is a live, AI-powered web app that helps users generate personalized gift ideas based on a recipient’s age, interests, and budget. Built with React, Supabase (for authentication and database), FastAPI, and the Gemini API, the app presents a variety of affordable and thoughtful options to reduce pressure around gift-giving. The backend is deployed on Render, while the frontend is hosted on Vercel. The system also supports user accounts and securely stores gift history using JWT authentication and Pydantic for data validation.

Zoopedia

Check it Out

Zoopedia is a live, daily animal guessing game built with Next.js, Redis (via Upstash), and the Gemini API. Users get AI-generated hints with each incorrect guess and have a limited number of attempts to identify the animal of the day. The game tracks user progress, awards points based on performance, and features a daily and all-time leaderboard. Hints are generated once per day and distributed consistently across all users. The frontend is hosted on Vercel, while Redis handles game logic and state persistence without requiring user login.

UP YOU GO

Check it out

Up You Go is a roguelike obby (obstacle course) where players face the ultimate test of precision and persistence with only one life per run. Each session challenges players to climb as high as possible through an increasingly difficult vertical obstacle course. Along the way, players collect coins that can be spent between runs to unlock skills, upgrades, and power-ups enhancing their chances of reaching new heights in future attempts.
